# Əlaqə

Layihə ilə bağlı hər hansı sualınız, təklifiniz və ya əməkdaşlıq imkanlarınız
varsa, layihənin idarəçiləri ilə əlaqə saxlaya bilərsiniz.

Əgər ümumi sualınız varsa,
[GitHub Discussions](https://github.com/BarishNamazov/kodlar.az/discussions)
bölməsində paylaşmağınız tövsiyə olunur. Beləliklə, sualınızın cavabı digər
istifadəçilərə də faydalı ola bilər. Sual verməzdən əvvəl mövcud müzakirələrə
göz atmağınız faydalı ola bilər.

Əgər daha spesifik və ya açıq müzakirəyə uyğun olmayan bir mövzu barədə yazmaq
istəyirsinizsə, idarəçilərlə birbaşa e-poçt vasitəsilə əlaqə saxlaya bilərsiniz:
[contact@kodlar.az](mailto:contact@kodlar.az).
